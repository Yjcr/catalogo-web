django-admin startproject "nombre del proyecto": este comando nos permite crear un proyecto de django
el archivo "manage.py" nos permitira gestionar nuestro proyecto, permitiendo ejecutar una multitud de comandos que 
nos facilitara el desarrollo del proyecto

python "manage.py" runserver: nos permite ejecutar la base de datos en cuestion
python "manage.py" runserver "puerto en cuestion": este comando nos permite cambiar de puerto a nuestra base de datos
python "manage.py" --help: este comando consulta toda los funciones que nos permitiran administrar nuestro proyecto

python "manage.py" startapp "nombre de la app": este comando nos permitira crea una app de nuestro proyecto
una vez creada la app, esta vendra con una serie de archivos que nos permitira especificar las diferentes funcionalidades de la aplicacion

inciso: una aplicacion o app en django es un componenete del proyecto que se encargara de realizar un funcion especifica
en el mismo
